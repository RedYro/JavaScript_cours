Variables JavaScript

Les variables en JavaScript sont comme des boîtes, on peut y stocker ce que l'on souhaite.

L'initialisation d'une variable se fait en déclarant un nom (que l'on choisit)

Déclaration de variables

  • const : permet de créer une constante accessible uniquement en lecture. Sa valeur ne peut être modifié par des réaffectations ultérieures, une constante ne peut être déclarée à nouveau.
  • let (anciennement "var") : sa valeur peut être changée contrairement aux constantes.

Exemples

  • Déclaration d'une variable avec const : const TEST = `Hello World !`;
  • Déclaration d'une variable avec let : let test1 = `Bye !`;
