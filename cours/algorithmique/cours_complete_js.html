Variables JavaScript

Les variables en JavaScript sont comme des boîtes, on peut y stocker ce que l'on souhaite.

L'initialisation d'une variable se fait en déclarant un nom (que l'on choisit)

Déclaration de variables

  • const : permet de créer une constante accessible uniquement en lecture. Sa valeur ne peut être modifié par des réaffectations ultérieures, une constante ne peut être déclarée à nouveau.
  • let (anciennement "var") : sa valeur peut être changée contrairement aux constantes.

Exemples

  • Déclaration d'une variable avec const : const TEST = `Hello World !`;
  • Déclaration d'une variable avec let : let test1 = `Bye !`;

Types de valeurs en JavaScript

Il existe plusieurs types de valeurs, que l'on peut assigner à nos variables :

Types :

  • chaînes de caractères (string) => phrase, entre ('') ("") (``)
  • nombres => 5, 10, 12.8
  • boolean => "true" ou "false"
  • array (tableau) => ["Julien", 10]
  • null
  • class
  • objets
  • function

Les différentes manières d'initialiser une chaîne de caractères

  • ("") double quotes : let variable = "test";
  • ('') simple quotes : let variable2 = 'test2';
  • (``) back quotes : let variable3 = `test3`;

En JavaScript, la convention de nommage est le CamelCase; chaque nouveau mot de notre variable doit être en majuscule.

Exemples :

  • let monTexte = "Voici une phrase écrite entre double quote"
  • let monTexte2 = 'Voici une phrase écrite dans une simple quote aujourd\'hui';
  • let monTexte3 = `Voici une phrase écrite aujourd'hui`;

Déclaration d'une variable avec une phrase en y insérant une constante :

  • let bienvenue = `Aujourd'hui c'est l'arrivée de Grégory`;
  • const PRENOM = `Grégory`;
  • bienvenue = `Aujourd'hui c'est l'arrivée de ${PRENOM}`;

Déclaration de variables avec des nombres :

  • let chiffre = 55;
  • let negatif = -51;
  • let decimal = 5.5;

Déclaration de variables de type "boolean" (seulement 2 valeurs possible : "true" ou "false") :

  • let choix1 = true;
  • let choix2 = false;

Boîtes de dialogues (non-considéreés comme des types)

    alert() permet d'afficher un message d'alerte, exemple :

  • alert(`Ceci est un message d'alerte !`);

    • confirm() permet de confirmer une action, exemple :

  • confirm(`Confirmez-vous ?`);

    • prompt() permet de récupérer une donnée de l'utilisateur, exemple :

  • prompt(`Quel est votre âge ?`);

Nombres en JavaScript

En JavaScript on peut effectuer tout type de calcul.

Exemples :

    Addition :

  • let addition = 10 + 5;

    • Soustraire :

  • let soustraction = 16 - 4;

    • Multiplication :

  • let multiplication = 5 * 4;

    • Division :

  • let division = 20 / 4;

    • Modulo :

  • let modulo = 10 % 3;

    • x², x^n (nombre au carré) :

  • let valeurAuCarree = valeur ** 2;

Raccourcis, incrémentation et décrémentation

    Il existe des raccourcis en JavaScript pour faire des calculs tels que :

  • let g = 10;
    g+= 5;     (équivaut à "g = 10 + 5" soit "g = 15")

    • L'incrémentation permet d'ajouter 1 à une variable contenant un nombre, exemple :

  • let h = 50;
    h++;
    (équivaut à "h = 50 + 1;" soit "h = 51;")

    • La décrémentation fait l'inverse, elle soustrait 1, exemple :

  • let i = 50;
    i--; (équivaut à "i = 50 - 1;" soit "i = 49;")

String en JavaScript

Les "strings" sont des chaînes de caractères et se déclare entre simple quotes '', double quotes "" et back quotes ``.

Conversion d'une chaîne de caractères en nombre (String => Number)

    Exemple :

  • let nombre = `24`;
    unNombre = parseFloat(nombre);

Conversion d'un nombre en chaîne de caractères (Number => String)

    Exemple :

  • unNombre = unNombre.toString();

Propriétés et méthodes des tableaux

  • "length": permet de connaître le nombre d'éléments dans un tableau

    • Exemple :

  • console.log(colors.length); => Affiche "4" (4 éléments)
  • "push" : permet d'ajouter un élément à la fin d'un tableau

    • Exemple :

  • colors.push(`Purple`);
    console.log(colors);     => Affiche le tableau avec la nouvelle valeur insérer à la fin "[`Red`,`Blue`,`Red`,`Blue`,`Purple`]"
  • "pop" : permet de supprimer le dernier élément d'un tableau

    • Exemple :

  • colors.pop();
    console.log(colors);     => Affiche le tableau sans la dernière valeur insérer précedemment avec "push()" "[`Red`,`Blue`,`Red`,`Blue`]"
  • "unshift" : permet d'ajouter un ou des éléments au début du tableau

    • Exemple :

  • colors.unshift(`White`,`Black`);
    console.log(colors);     Affiche le tableau avec les 2 nouvelles valeurs insérer au début "[`White`,`Black`,`Red`,`Blue`,`Red`,`Blue`]"
  • "splice" : permet d'ajouter, supprimer ou remplacer des éléments n'importe où dans un tableau

    • "splice" prend 3 arguments :

  • position de départ : à partir d'où commence le changement
  • nombre d'éléments à remplacer ou enlever
  • élément(s) à ajouter au tableau

    • Exemple :

  • colors.splice(1, 0, `Brown`, `Orange`);
    console.log(colors);     Affiche "[`White`,`Brown`,`Orange`,`Black`,`Red`,`Blue`,`Red`,`Blue`]"
  • color.splice(2, 1, `Pink`);
    console.log(colors);     Affiche "[`White`,`Brown`,`Pink`,`Black`,`Red`,`Blue`,`Red`,`Blue`]"
  • "slice" : permet de découper un tableau et d'en créer un autre

    • Exemple :

  • let joursSemaines = [`Lundi`,`Mardi`,`Mercredi`,`Jeudi`,`Vendredi`,`Samedi`,`Dimanche`];
    let joursTravail = joursSemaine.slice(0, 5);
    console.log(joursTravail);     Affiche "[`Lundi`,`Mardi`,`Mercredi`,`Jeudi`,`Vendredi`]"
    let joursRepos = joursSemaine.slice(5, 7);
    console.log(joursRepos);     Affiche "[`Samedi`,`Dimanche`]"
  • "join" : permet de choisir un séparateur entre les éléments d'un tableau en concaténant les différentes valeurs de celui-ci

    • Exemple :

  • let concatColors = colors.join(` - `);
    console.log(colors);     Affiche "White - Brown - Pink - Black - Red - Blue - Red - Blue" avec un document.write() par exemple
  • "includes" : permet de déterminer si un tableau contient une valeur précise, à noter quelle est sensible à la casse c'est-à-dire que la méthode vérifiera par exemple l'écriture d'une chaîne de caractères, si on écrit "html" au lieu de "HTML" celle-ci renverra "false" (méthode qui renvoie un résultat "boolean")

    • Exemple :

  • let technologies = [`HTML`,`CSS`,`JS`];
    let verif = technologies.includes(`CSS`);     Affiche "true" car le tableau contient bien la "CSS"

  • "concat" : permet de fusionner différents tableaux entre eux afin d'en créer un nouveau

    • Exemple :

  • let totalTableaux = technologies.concat(colors, myArray, array4Dimensions);
    console.log(totalTableaux);     Affichera toutes les valeurs des précédent tableaux dans un seul tableau

Conditions en JS

Opérateurs logiques

Opérateurs de comparaison pour structure traditionnelle

  • ">" : strictement supérieur
  • "<" : stritement inférieur
  • ">=" : supérieur ou égal
  • "<=" : inférieur ou égal
  • "==" : comparaison de valeur, vérifie si 2 valeurs de 2 variables sont identiques
  • "===" : comparaison de valeur ET de type // signifie "STRICTEMENT ÉGAL À" // vérifie la valeur et le type (string, number, etc.)
  • "!=" : différent "de" en valeur
  • "!==" : strictement différent de (valeur et type)

    Exemples :

  • let nb1 = 1;
    let nb2 = 2;
    console.log(nb1 > nb2);     Affiche "false"
  • console.log(nb1 < nb2); Affiche "true"
  • console.log(nb1 >= nb2); Affiche "false"
  • console.log(nb1 <= nb2); Affiche "true"
  • let nb3 = 3;
    let nb3Bis = `3`;
    console.log(nb3 == nb3Bis);     Affiche "true" car les 2 variables possèdent toutes deux la même valeur
  • console.log(nb3 === nb3Bis); Affiche "false" car même si les 2 variables ont la même valeur, elles ne possèdent pas le même type (nb3 = int ; nb3Bis = string)

    Opérateurs logiques "&&" "||" "!"

  • "&&" = AND : permet d'évaluer une comparaison entre des variables et de renvoyer le résultat en "boolean" (true ou false), si toutes les expressions sont testées "true" alors on aura comme résultat "true" et si une des expressions est testée "false" le résultat sera "false"
  • "||" = OR : permet d'évaluer une comparaison de variables, si 1 ou 2 expressions sont "true" alors le résultat sera "true", si 2 expressions sont "false" alors le résultat sera "false"
  • "!" = NOT : permet de transformer "true" en "false" et vice-versa

    Exemples :

    Opérateur && :

  • true && true; Affiche "true"
  • true && false; Affiche "false"
  • false && true; Affiche "false"
  • false && false; Affiche "false"

    • Opérateur || :

  • false || false; Affiche "false"
  • true || false; Affiche "true"
  • true || true; Affiche "true"
  • false || true; Affiche "true"

    • Opérateur ! :

  • !true; Affiche "false"
  • !false; Affiche "true"

Structure conditionnelle if() else else if()

    Exemples :

  • let a = 10, b = 5, c = 2;
    if(a > b){ // si a > b => "true" alors le code dans le "if" s'exécute
    console.log("a" est supérieur à "b");
    } else{ // sinon si "false" le code dans le "else" s'exécutera
    console.log("a" est inférieur à "b");
    }

    • if avec &&

  • if((a > b) && (b > c)){ // si a > b et b > c => "true" alors le code dans le "if" s'exécute
    console.log("a" est supérieur à "b");
    } else{ // sinon si une des conditions du if est "false" le code dans le "else" s'exécutera
    console.log("a" est inférieur à "b");
    }

    • if avec ||

  • if((a > b) || (b > c)){ // si a > b ou b > c => "true" alors le code dans le "if" s'exécute
    console.log("a" est supérieur à "b");
    } else{ // sinon si les 2 conditions du if sont "false" le code dans le "else" s'exécutera
    console.log("a" est inférieur à "b");
    }

    • if else if else

  • if(a == 8){ // false
    console.log("a" est égal à 8);
    } else if(a != 10){ // false
    console.log("a" est différent de 10);
    } else{ // true
    console.log("a" est égal à 10);
    }
  • if(a == 10){ // true
    console.log("a" est égal à 10);
    } else if(a > b){ // true
    console.log("a" est supérieur à "b");
    } else{
    console.log("a" est "false");
    }

    Condition ternaire

    La condition ternaire nous permet de faire une vérification mais seulement sur une ligne

    Exemple :

  • let meteo = `pluie`;
    console.log((meteo == `pluie`) ? `Prenez un parapluie` : `Rangez le parapluie`);     Affiche "Prenez un parapluie"
  • ? => if
  • : => else

    Condition switch

    La condition switch répond à des cas bien spécifiques, on doit écrire chaque condition à l'intérieur d'un "case" et sortir de celui-ci avec un "break" (stop la condition)

    Exemple :

  • let valeur = 5 + 3;
    switch(valeur){
    case 5 :
    console.log(`La valeur est égale à 5`);
    break;
    case 8 :
    console.log(`La valeur est égale à 8`);
    break;
    case 12 :
    console.log(`La valeur est égale à 12`);
    break;
    default :
    console.log(`La valeur est indéterminée`);
    break;
    };     Affiche le code dans le "case 8" car la valeur de la variable "valeur" est égale à 8

Boucles en JavaScript

    Boucle : "FOR"

    for prend 3 arguments :

  • initialisation
  • condition de continuation
  • incrémentation

    • Syntaxe :

  • for(initialisation; condition de continuation; incrémentation){
    // Code à exécuter
    };

    • Exemple 1 :

  • for(let i = 0; i <= 10; i++){
    console.log(`Instruction exécutée : ${i}`);
    };     Affiche 10 fois "Instruction executée : 1" etc jusqu'à 10 car i <= 10
  • "i = 0" => initialisation du départ de la boucle à 0
  • "i <= 10" => tant que la valeur de i est inférieure ou égale à 10
  • "i++" => incrémentation d'un tour dans la boucle

    • Exemple 2 :

  • for(let i = 0; i <= 10; i+=2){
    console.log(`Compte 2 par 2 : ${i}`);
    };

    • Exemple 3 : Array

  • let users = [`Yro`,`Hada`,`Yato`,`Lya`];
    for(let i = 0; i < users.length; i++){
    console.log(users[i]);
    };     Affiche le contenu du tableau

    Boucle : "WHILE"

    Syntaxe :

  • "while" => expression qui vaut "true" ou "false"
  • while(expression "true" ou "false"){
    // Tant que l'expression = "true", code à exécuter
    };

    • Exemple :

  • let i = 0;
    while(){
    console.log(i + `----`);
    i++;
    }
  • "let i = 0" initialisation d'une variable à 0
  • "while(i <= 5)" Tant que "i" est inférieur ou égal à 5
  • "i++" incrémentation

Fonctions en JavaScript

    Fonctions prédéfinies et utilsateurs

    JavaScript possède des fonctions "function" qui sont des morceaux de codes encapsulés que l'on appel au besoin pour exécuter une action précise.

    On distingue 2 types de fonctions : prédéfinies & utilisateurs

Fonctions prédéfinies sur des nombres

    "toFixed()" : définit un nombre de chiffre à afficher après la virgule et retourne le résultat en chaîne de caractères.

    "Math.pow()" : permet de faire des calculs avec des puissances. (2² par exemple)

    "Math.PI" : permet d'obtenir le nombre PI avec 15 chiffres après la virgule.

    "Math.floor" : permet d'obtenir le nombre entier inférieur.

    "Math.max" : permet de retourner la valeur la plus haute.

    "Math.min" : permet de retourner la valeur la plus basse.

    "Math.random()" : retourne un nombre décimal entre 0 et 1 qui change à chaque rafraîchissement de page, le nombre est totalement aléatoire.

    Exemples :

  • toFixed() :
    let testNumber = 1.5555555;
    let number = testNumber.toFixed(2); console.log(number);     Affiche "1.56"
  • Math.pow() :
    let testMath = Math.pow(2,3);
    console.log(testMath);     Affiche "8"
  • Math.PI :
    let MathPI = Math.PI;
    console.log(MathPI);     Affiche "3.141592653589793"
  • Math.floor() :
    let MathFloor = Math.floor(1.9);
    console.log(MathFloor);     Affiche "1"
  • Math.max() :
    let MathMax = Math.max(10,8,14,50,6);
    console.log(MathMax);     Affiche "50"
  • Math.min() :
    let MathMin = Math.min(10,8,14,50,6);
    console.log(MathMin);     Affiche "6"
  • Math.random() :
    let MathRandom = Math.random();
    console.log(MathRandom);     Affiche un nombre décimal aléatoire changeant à chaque rafraîchissement

Fonctions prédéfinies sur des chaînes de caractères

    ".length" : permet de retourner le nombre de caractères présents dans une chaîne de caractères

    ".indexOf()" : permet de retourner la position d'un caractère dans la chaîne (position de l'argument passé en paramètre)

    ".toLowerCase()" : permet de retourner la chaîne de caractères en minuscule

    ".toUpperCase()" : permet de retourner la chaîne de caractères en majuscule

    ".trim()" : permet de supprimer les espaces avant et après la chaîne de caractères

    ".split()" : permet de retourner un tableau d'une chaîne de caractères (string to array)

    Exemples :

  • .length :
    let answer = `Hello`;
    console.log(answer.length);     Affiche "5" (nombre de caractères présent dans la chaîne)
  • .indexOf() :
    let answer = `Hello`;
    console.log(answer.indexOf(`l`));     Affiche "2" (position du caractère dans la chaîne)
  • .toLowerCase() :
    let answer = `HELLO`;
    console.log(answer.toLowerCase());     Affiche "hello"
  • .toUpperCase() :
    let answer = `hello`;
    console.log(answer.toUpperCase());     Affiche "HELLO"
  • .trim() :
    let answer = ` some text `;
    console.log(answer.trim());     Affiche "some text" au lieu de " some text "
  • .split() :
    let words = `metro, bus, train`;
    console.log(words.split(`,`));     Affiche "['metro', ' bus', ' train']", la virgule dans "split(`,`)" sert de séparateur entre chaques mots

Fonctions utilisateurs

    Un des grands principes en programmation est d'avoir un code dit "DRY" (Don't Repeat Yourself).

    Les fonctions sont des blocs de codes réutilisables et modifiables à volonté.

    Exemples :

  • function myFunction(){
    console.log(`My first function`);
    }
    myFunction(); // Exécution de la fonction     Affiche "My first function"
